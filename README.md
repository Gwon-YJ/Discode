# 🎨 Discord 배경 꾸미기 가이드

디스코드 배경을 꾸미는 방법은 **공식 기능(안전)** 과  
**비공식 클라이언트(Vencord / BetterDiscord)** 를 사용하는 방법이 있습니다.

---

## 1️⃣ 공식 기능으로 꾸미기 (안전 👍)

> ⚠️ 서버/채팅창 배경 이미지는 **공식적으로 불가능**

### ✅ 프로필 배너 & 테마 (Discord Nitro 필요)

#### 🔹 프로필 꾸미기
- `설정 → 프로필`
- **프로필 배너**
  - 이미지 또는 GIF 업로드 가능
- **아바타 데코레이션** 사용 가능

#### 🔹 외형 설정
- `설정 → 접근성 / 외형`
- 테마
  - 🌙 다크
  - ☀️ 라이트
  - 🌑 미드나잇
- 채팅 밀도 조절
- 폰트 크기 및 가독성 설정

✔️ 공식 기능이므로 **계정 제재 위험 없음**

---

## 2️⃣ Vencord / BetterDiscord로 배경 꾸미기 (PC 전용 ⚠️)

> ⚠️ **비공식 클라이언트** 사용  
> 계정 제재 가능성은 낮지만 **존재함**

### 🔹 준비물
- PC (Windows / macOS)
- **Vencord** *(본 가이드는 Vencord 기준)*
- 테마 파일 (`.theme.css`)

### 🔹 설치 및 적용 방법
1. Vencord 설치
2. 디스코드 실행
3. `설정 → Themes`
4. 테마 다운로드 후 적용
5. 배경 이미지 및 투명도 커스터마이즈

### 🔹 인기 배경 테마
- **ClearVision** ⭐ (가장 유명)
- **Frosted Glass**
- **Dark Matter**
- **Transparent Discord**

✔️ 변경 가능 영역
- 채팅창 배경
- 서버 목록
- 전체 디스코드 배경

---

## 3️⃣ 모바일 환경

- ❌ 배경 이미지 변경 불가
- ✅ 프로필 배너 / 테마만 가능 (Nitro 필요)
